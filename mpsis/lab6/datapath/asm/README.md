Итоговая последовательность выполнения:
00: addi x1, x0, 0x75C

04: addi x2, x0, 0x8A7

08: add x3, x1, x2

0C: and x4, x1, x2

10: sub x5, x4, x3

14: mul x6, x3, x4 → ошибка, нет поддержки mul

18: jal x15, 0x50 → переход на 0x50

Пропущены: 1C–4C

50: lbu x15, 0x0(x4)

54: auipc x16, 0x00004

58: bne x3, x4, 0x08 → переход на 0x60

60: jal x17, 0x00004

64: jalr x14, 0x0(x17) → бесконечный цикл
